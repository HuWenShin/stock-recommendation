# Stock Recommendation System
Analyze news title keywords to recommend stock <br>

### Sample 1
**input:** <br>
/assisting_data/news_title.txt <br>
/assisting_data/news_dict.txt <br>
/assisting_data/company_category.txt <br>
半導體股,12,4:2:1 <br>
<br>
**output:** <br>
日月光購買8張 <br>
聯發科購買3張 <br>
台積電購買1張 <br>
